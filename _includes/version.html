API V1 Introduction

MercadoLibre Developers Program recomend use of API V1 This is the most stable version, and it will not change. Any change made to this version will improve performance and bugs fixes. All changes will be notify in this page.

Overview
Mercado Libre recomienda el uso de la API V1 para desarrollo de aplicaciones. Garantiza que la misma no va a sufrir cambios de improviso, y en el caso que así sea, se comunicará desde este sitio. La API V1 es la version estable de la api.developers.mercadolibre.com La api.developers, puede sufrir cambios de improviso, sin necesidad de informar a la comunidad de developers, por eso recomendamos fuertemente el uso de V1. Detalles de Diseño (Design Considerations) La API v1 mantendrá los estándares de diseño establecidos. ML asegura que bajo ningún concepto se cambiará el formato de respuesta de los diferentes recursos de la API.

Consideraciones claves:
Todas las respuestas serán codificadas con JSON. JSONP también será soportado. Todos los requests/responses codificación UTF-8 Las fechas respetan la codificación 8601 Todos los errores tiene un formato establecido La data de v1 es la misma que la api.developers Por más información ver (design considerations) API Directory Pueden encontrar aquí todos los recursos de la API versión 1 detallados.

Control de Cambios
Se detallara en esta seccion los cambios que vallan ocurriendo entre cambios de version, por ejemplo metodos nuevos, cambios de metodos. Cuando la api V1, evolucione a una V2 o VX, los cambios se iran registrando en el site. Recordamos que nos comprometemos a informar con antelacion a un cambio.

How to connect?
De la misma manera que usted lo hacía habitualmente:

$ curl https://api.mercadolibre.com/sites/MLA/search?q=ipod
Antes: API Developers
Ahora con v1: 
$ curl https://v1.api.mercadolibre.com/sites/MLA/search?q=ipod


El resultado es el mismo.
